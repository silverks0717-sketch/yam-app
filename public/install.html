这里既是 YAM 的下载入口，也是发给朋友就能直接打开的官网。安卓可以下载 APK，iPhone / iPad 可以直接用 Web 或添加到主屏幕。

固定下载地址 /downloads/latest.apk
安装步骤

1. 下载 APK。2. 在手机里点开。3. 第一次安装时允许此来源安装。4. 确认安装。5. 打开使用。

更新方式

每次打开 App 或 Web 版都会做版本检查；安卓会跳到最新 APK，iPhone / iPad 会直接进入最新 Web / PWA。

Android

适合华为 / 安卓手机

第一阶段优先照顾安卓下载安装路径，不走 App Store 审核流程。

Web 版

不用安装也能直接使用

打开网页就能注册、登录、记录和看图表，适合先试用或临时从电脑继续记录。

iPhone / iPad

Safari 里可添加到主屏幕

不走 App Store。用 Safari 打开首页，点“添加到主屏幕”，就能像 App 一样使用 Web 版。

更新方式

稳定下载链接
